Débugger avec une transformation simple

On fait une transformation simple et bien visible avec jQuery. et on déplace ce court script pour voir à quel niveau le code a planté.

$("h1").css("color","red") ;

Avantage : simple

Inconvénients : donne peu d'information et transforme la page (et fastidieux si plusieurs erreurs)

Débugger avec la fonction alert

Affiche du contenu textuel dans une pop-up 

alert("Hello") ; alert(maVariable)

Avantage : visible

Inconvénients : intrusif et agaçant à la longue

Débugger avec la console JS

La console JavaScript est présente dans les dernières version de Firefox et Chrome de manière native

même fonctionnement que alert() mais pas de pop-up

console.log($(h1)) ; console.log("hello")

Avantages : invisible, possibilité de multiples débuggages d'un coup

mieux encore la console permet des points d'arrêts, espions, etc.

Pour les anciens Firefox, Firebug permet une communication entre le script et la console.
